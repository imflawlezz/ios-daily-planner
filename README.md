# DailyPlanner

University assignment project for an iOS lab course.

This app is a **UI/layout exercise** only. It presents a form that looks like a daily planner (name, email, task, priority, reminder toggle) but **does not implement** any real planning or reminder functionality. There is no persistence, no notifications, and no calendar integration.

Use it to try out storyboard layouts, form controls, and basic validation — not as an actual planner or reminder app.

## Requirements

- Xcode
- iOS deployment target (see project settings)

## Building & running

1. Open `DailyPlanner.xcodeproj` in Xcode.
2. Select a simulator or device and run (⌘R).

## What it does

- Displays a form with text fields, segmented control, switch, and date picker.
- Validates required fields and shows alerts.
- Shows a summary alert on “Save” (no data is saved).
- **Does not:** save entries, set reminders, or integrate with Calendar/Reminders.
